## 题目描述

我们可以用2*1的小矩形横着或者竖着去覆盖更大的矩形。请问用n个2*1的小矩形无重叠地覆盖一个2*n的大矩形，总共有多少种方法？

## 解题说明

跟第 7 题和第 8 题一样，都是斐波那契数列的变形题。

对于最右下角的格子

- 如果那里的小矩形是竖着放的，那么问题就转变成了 `2*(n-1)` 的矩形
- 如果是横着放的，那么上面两格也必须是横着放，问题就编程了 `2*(n-2)` 的矩形

所以 `dp[n] = dp[n - 1] + dp[n - 2]`。
